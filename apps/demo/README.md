# Rice AI Components Demo

这是 Rice AI 组件库的演示应用，展示了组件库中各种组件的用法和效果。

## 开发

```bash
# 安装依赖
pnpm install

# 启动开发服务器
pnpm dev

# 构建
pnpm build
```

## 特性

- 🚀 基于 Next.js 14
- 📦 使用 @rice/react 组件库
- 🎨 Tailwind CSS 样式
- 📱 响应式设计
- 🔧 交互式组件演示

## 组件展示

### Chat 组件
- 可自定义标题
- 可控制显示/隐藏
- 支持点击事件
- 可自定义样式
